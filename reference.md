    api_key="YOUR_API_KEY",
)

    id=1,
    next_token="eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9",
    query="my search query",
)

```
</dd>
</dl>
</dd>
</dl>

<dl>
<dd>

<dl>

</dd>

</dd>
</dl>

<dl>
<dd>

**relevance:** `typing.Optional[float]` ‚Äî The minimum search relevance score required to include the result. By default, this is 10.0.
